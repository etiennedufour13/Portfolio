Plateau procédural

game design

unity

travail audio

Dans le cadre de mon mémoire de DNMADE à l’ENSAAMA, j’ai exploré le potentiel du jeu hybride. J’ai réalisé une série d’expériences qui avaient pour but d’explorer les possibilités apportées par le numérique dans le contexte d’une expérience ludique tangible.

Ce projet met en avant le potentiel apporté par la génération procédurale dans le cadre d’une expérience ludique tangible. J’ai développé une application permettant, à partir de paramètres modulables, de générer un terrain unique pouvant être ensuite installé sur le plateau physique.
